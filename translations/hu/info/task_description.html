Adott egy tömb, amelynek két egész eleme van. Adja össze ezt a két értéket!

Bemeneti adatok: egy kételemű lista (list). Minden listaelem egy egész szám (int).

Kimeneti adatok: a két szám összege egész számként (int).

Példák: 

checkio([5, 5]) == 10
checkio([7, 1]) == 8
checkio([5, 5]) == 10
